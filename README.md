#### Setup Instructions:

**1. Clone this repo**

```
git clone https://github.com/Arioum/uifry-assessment.git
```

**2. Do the following steps:**

```
cd uifry-assessment
npm install
```

**3. Open the current folder in a code editor (ex: vs code)**

```
code .
```

**4. Start the app:**

```
npm run dev
```

**5. Click the link in the terminal or navigate to:**
`localhost:5173`
